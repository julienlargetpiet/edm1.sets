see_diff_all

Description

Allow to perform the opposite of intersect function to n vectors.

Usage

see_diff_all(...)

Arguments

...

are all the input vectors

Examples


vec1 <- c(3:6)
vec2 <- c(1:8)
vec3 <- c(12:16)

print(see_diff_all(vec1, vec2))

[1] 1 2 7 8

print(see_diff_all(vec1, vec2, vec3))

[1]  3  4  5  6  1  2  7  8 12 13 14 15 16
